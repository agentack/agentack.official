## Primary Brand Colors (Use These Most)

### Agentack Green — Main Brand Color
```
#00FF88
```
- **Usage**: Logo, primary CTAs, key highlights, icons, accents
- **Social Media**: Profile pictures, post highlights, story accents, link buttons
- **Psychology**: Innovation, technology, growth, energy

### Electric Teal — Secondary Accent
```
#00E5CC
```
- **Usage**: Secondary accents, hover states, badges, tags
- **Social Media**: Secondary elements, text highlights, borders
- **Psychology**: Modern, tech-forward, trustworthy

```

## Social Media Branding Guidelines

### Profile Picture / Logo
- **Background**: `#000000` or `#080808`
- **Logo/Text**: `#00FF88`

### Post Templates
```
Background:    #000000 or #080808
Headlines:     #F0F2F0
Body Text:     #8A9A94
Accents:       #00FF88 or #00E5CC
Borders:       #3A4A42
```

### Story Highlights
- **Icons**: `#00FF88` on `#000000`
- **Covers**: `#00FF88` accents on dark background

### Call-to-Action Buttons
- **Background**: `#00FF88`
- **Text**: `#001A0D` (dark green for contrast)

### Quote Cards / Testimonials
```
Background:     #111714
Quote Text:     #F0F2F0
Accent Border:  #00FF88 (left border)
Author Name:    #00E5CC
```

## Recommended Color Combinations

### ✅ High Contrast (Best for Readability)

Background: #000000
Text:       #F0F2F0

```

Background: #111714

## Accessibility

All color combinations meet WCAG AA standards:

| Combination | Contrast Ratio | Status |
|-------------|----------------|--------|
| Agentack Green on Black | 11.3:1 | ✅ Pass |
| Sage Mid on Black | 5.8:1 | ✅ Pass |
| Frost White on Black | 15.2:1 | ✅ Pass |
| Electric Teal on Black | 10.1:1 | ✅ Pass |

**Minimum required**: 4.5:1 for normal text (WCAG AA)
